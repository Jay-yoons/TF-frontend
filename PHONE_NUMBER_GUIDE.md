# 전화번호 입력 형식 개선 가이드

## 📱 개선된 전화번호 입력 시스템

### 🎯 목표
사용자가 더 편리하게 전화번호를 입력할 수 있도록 다양한 형식을 지원합니다.

### ✅ 지원하는 입력 형식

#### 1. **간단한 숫자 형식 (권장)**
```
01012345678
010-1234-5678
010 1234 5678
```

#### 2. **국제 형식**
```
+82 10 1234 5678
+821012345678
```

#### 3. **기존 형식**
```
82 10 1234 5678
821012345678
```

### 🔄 자동 변환 규칙

#### 입력 → 저장 형식
- **입력**: `01012345678`
- **저장**: `+82 10 1234 5678`

#### 저장 → 표시 형식
- **저장**: `+82 10 1234 5678`
- **표시**: `010-1234-5678`

### 📋 지원하는 전화번호 유형

| 유형 | 예시 | 설명 |
|------|------|------|
| 휴대폰 | `010-1234-5678` | 가장 일반적인 형식 |
| 서울 | `02-1234-5678` | 서울 지역번호 |
| 지역 | `031-123-4567` | 기타 지역번호 |

### 🛠️ 기술적 구현

#### 프론트엔드 (Vue.js)
- **파일**: `src/utils/phoneFormatter.js`
- **기능**: 실시간 포맷팅, 유효성 검사, 표시용 변환

#### 백엔드 (Spring Boot)
- **파일**: `PhoneNumberUtil.java`
- **기능**: 전화번호 정규화, 중복 검사, 형식 변환

### 🎨 사용자 경험 개선

#### 1. **입력 편의성**
- 사용자는 `01012345678`과 같이 간단하게 입력
- 자동으로 하이픈(-) 추가
- 11자리 제한으로 오타 방지

#### 2. **표시 개선**
- 마이페이지에서 `010-1234-5678` 형식으로 표시
- 읽기 쉬운 형식으로 변환

#### 3. **데이터 일관성**
- 모든 전화번호가 `+82` 국제 형식으로 저장
- 중복 검사 시 정규화된 형식 사용

### 🔧 개발자 가이드

#### 새로운 전화번호 입력 필드 추가 시
```javascript
import { formatPhoneInput, isValidPhoneNumber } from '@/utils/phoneFormatter';

// 입력 시 실시간 포맷팅
<input 
  v-model="phoneNumber" 
  @input="phoneNumber = formatPhoneInput($event.target.value)"
  placeholder="01012345678"
/>

// 유효성 검사
if (!isValidPhoneNumber(phoneNumber)) {
  alert('올바른 전화번호를 입력해주세요.');
}
```

#### 백엔드에서 전화번호 처리 시
```java
import com.restaurant.reservation.util.PhoneNumberUtil;

// 전화번호 정규화
String normalizedPhone = PhoneNumberUtil.normalizePhoneNumber(inputPhone);

// 유효성 검사
if (!PhoneNumberUtil.isValidPhoneNumber(inputPhone)) {
    throw new RuntimeException("올바르지 않은 전화번호 형식입니다.");
}
```
